# Article Insert (Contao)

Ein einfaches Frontend-Modul für Contao, mit dem die **Inhaltselemente eines bestehenden Artikels** in ein anderes Layout oder eine andere Seite eingebunden werden können.

## Funktionen
- Auswahl eines bestehenden Artikels
- Ausgabe aller veröffentlichten Inhaltselemente dieses Artikels
- Eigenes Frontend-Modul
- Eigenes Template (`mod_article_insert`)

## Kompatibilität
- Contao 4.13+
- PHP 8.0+

## Installation
Über den Contao Manager oder per Composer:

```bash
composer require article/insert
